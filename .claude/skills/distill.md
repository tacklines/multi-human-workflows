# /distill -- Synthesize Findings

Reduce a collection of raw findings into actionable insights.

## Usage

`/distill <findings-or-file>`

## Process

1. Read the input findings (from /gather output or a file)
2. Group by theme
3. Identify patterns, contradictions, and gaps
4. Produce a concise summary

## Output

Themed summary with 3-7 bullet points. Flag any contradictions or gaps.
